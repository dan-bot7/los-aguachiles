# Editar contenido

Los textos comerciales, platillos, precios, sucursales, reseñas y número de WhatsApp están en:

```txt
packages/shared/index.ts
```

Para cambiar el número de WhatsApp, edita:

```ts
export const whatsappNumber = "529841234567";
```

Para cambiar precios o nombres de platillos, edita el arreglo `featuredDishes`.
